# Data Connect

[Data Connect](https://www.sheffield.ac.uk/data-connect) is a University of Sheffield service to improve researcher access to health and social care data enhancing research capability and driving innovation. We work with the people and service providers of South Yorkshire to improve health outcomes across the region.

# CUREd+

Repositories for the [CUREd Plus](https://www.sheffield.ac.uk/scharr/research/centres/cure/projects/cured-how-access-data) project linking routine 
NHS data with Urgent Emergency Care (UEC) data in the Yorkshire and Humber region.
